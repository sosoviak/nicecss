    when-inside

    Since version — <p>0.1</p>
    @mixin when-inside($selector) { ... }

    Description

    Este mixin es un wrapper que sirve para definir nuevas propiedades que se harán activas si este elemento tiene como padre un elemento con el selector definido

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $selector

    Selector of parent element

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .element {
  @include when-inside('.active'){
    display: none;
  }
}
